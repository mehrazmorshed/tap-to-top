Tap the button and scroll to top immidiately.

== Description ==

This plugin creates a button in the bottom-left of the website and scrolls to top when the button in clicked.

== Installation ==

Upload the plugin, activate it, now you're done!

== Frequently Asked Questions ==

= How to enable tap to top button? =

Activate the plugin. That's it.

= How to disable tap to top button? =

Just deactivate the plugin.
